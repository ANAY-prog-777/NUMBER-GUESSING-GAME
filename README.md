Yo, welcome to my  game. I built this in like, 1.5-2 h just to see if I could make the computer do what I want. It is a classic guess the number game, but I added a Skull operator because, well, why not?

--> How it works-----------------------------
It’s simple, don't overthink it.

You run the code (python guesser.py).

The computer picks a number between 1 and 100.

You gotta guess it in 7 tries before the Skull gets hungry.

I added hints so you know if you're hitting the "Underground" (too low) or going "Space" (too high).

--> Why I built this   -------------------------------------
I wanted to level up my Python game. I messed around with while loops and if/elif logic to make sure the game doesn't just crash when you do something wrong. It's not fancy, but it's my code, and it actually works without any of that complicated def function stuff.

--->How to run this cortisol lowering game  ------------------------------------------>
You need Python installed, obviously.

Open your terminal.

Get into the folder where you saved the file.

Type python guesser.py and hit enter.

If it breaks, that's on you (jk, just check your indentation). ,SOMETIMES THERE MAYBE A TERIMINAL ERROR TRY TO RUN THE CODE TWICE AND PRAY
